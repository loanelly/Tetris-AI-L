# Tetris-AI-L
Tetris-AI-L.py — эволюционный ИИ для Тетриса на базе Simple Edition. Вместо слепого пошагового RL, робот использует планировщик Dellacherie и Lookahead-предиктор, просчитывая доску на 2 шага вперед. Генетический алгоритм мутирует веса высоты, дыр и шероховатости поля, пробивая потолок в 100000+ очков. Поддерживает 3 режима скорости.
